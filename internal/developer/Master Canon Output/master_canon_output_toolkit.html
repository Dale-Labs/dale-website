DALE Master Canon Output

CTS-compliant canonical record wrapper for CoS, ECOA, ARA, Runner, SEC, CBC, CTS, and observed outcomes.

Version1.0.0
Standard BasisCTS / URT v1.0.0
Schemamaster_canon_output_schema.json
Specmaster_canon_output_spec.yaml

Purpose

The master canon output preserves each canon's local output while adding the CTS fields needed for shared traceability, runtime interpretation, governance attribution, temporal continuity, contradiction visibility, missingness visibility, and reconstruction.

Required Record Shape

Required

cts_trace

The persistent CTS identity and lineage envelope.

Required

canon_context

Source canon, version, stage, format, upstream, and downstream context.

Mapping

urt_mapping

Local state preserved and mapped into the shared URT runtime language.

Required

continuity

Dependency, adaptation, outcome, temporal, reconstruction, and propagation fields.

Required

contradictions

Visible structural tensions. Empty only when explicitly verified.

Required

missingness

Visible uncertainty, absence, conflict, inaccessible evidence, or unknown origin.

Required

local_record

The original canon-specific output. It is preserved, not overwritten.

Required

validation

CTS compliance result and open corrections.

Canon Corrections

CanonCorrection Required
COSAdd root and parent lineage, governance, confidence, authority, URT state mapping, and explicit missingness.
ECOAAdd root and parent lineage, governance, confidence, authority, normalized absence conditions, and URT structural-state mapping.
ARAAdd CTS trace envelope, normalized contradictions, URT regime-state mapping, governance, confidence, reconstruction, and propagation.
RUNNERCreate missing YAML and HTML formats. Enforce CTS runtime, adaptation, temporal, and reconstruction fields.
SECAdd CTS trace envelope. Map RES-* states to URT. Preserve unresolved conditions as missingness or contradictions.
CBCAdd CTS trace envelope. Map governance coherence states to URT. Preserve governance flow tensions and accessibility reconstruction.
CTSCreate missing YAML and HTML formats. Publish this envelope as the shared implementation layer.
